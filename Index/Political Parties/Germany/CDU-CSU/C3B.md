# CDU-CSU – Cultural 3B (Xenophobia)

**Score: 42**

### CDU/CSU Final Score: 3A 0 | 3B 42

The CDU/CSU maintains open access while advocating moderate but consistent cultural expectations for membership.

#### Allocation Mechanisms Explained Analysis
- Strong support for the concept of *Leitkultur* as a guiding framework for integration and naturalisation.
- Defends language requirements, citizenship tests, and values commitments as conditions for becoming German.
- Opposes overly rapid or low-threshold naturalisation, arguing that citizenship should reflect success in integration.
- Criticises the 2024 citizenship reform as weakening the link between membership and cultural alignment.
- Continued emphasis that Germany is an immigration country, but one that requires newcomers to adopt core German values and norms.

The CDU/CSU treats legal status as the gateway to membership but insists that cultural integration should be meaningful and that shared cultural alignment with German values and norms should be required for full Endos status. This produces a moderate 3B score.
