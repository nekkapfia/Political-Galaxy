# 02 - Distinct Phenomena

**Full Universal Rule:**

Each axis must measure a genuinely distinct phenomenon. Axes that collapse into near-identical measurements or highly correlated outcomes are merged or eliminated. Semantic similarity is watched carefully.

**Key Application Guidelines:**

- No two axes should be measuring essentially the same thing under different names.
- High correlation between axes is allowed only if they capture meaningfully different aspects (e.g., personal vs economic autonomy).
